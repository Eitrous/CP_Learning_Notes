#### *cin* 和 *cout*

*尽量不要使用 `endl`[^1]，改用 `'\n'`*[^2]

### 常量设置

#### 尽量用`const`而非`define`

以下两种写法输出结果不同：

```cpp
#define N 1e5 + 10
int main()
{
    cout << N * 2 << '\n';//输出100020
    return 0;
}
```

```cpp
const long long N 1e5 + 10
int main()
{
    cout << N * 2 << '\n';//输出200020
    return 0;
}
```

***原因：`#define`只是简单的宏替换***

也可以使用

```cpp
#define N (1e5 + 10)
```

不过还是尽量用`const`吧

### 开数组

#### 尽量开全局数组

原因：
1. 自动初始化为0
2. 占用堆空间，可以开大数组

# 算法篇

## 前缀和与差分

### 前缀和
特点：
1. 简化计算
2. 要求运算方法具有可加性，如加法、乘法、异或

### 差分
特点：
1. 可通过前缀和还原
2. 后缀区间修改
3. 静态的，不能边询问边修改
4. 全部修改完成后再询问可达到 *O(1)* 复杂度
